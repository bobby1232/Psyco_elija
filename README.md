# Psyco Elija Bot

Бот читает групповой чат и занимается просветительской работой с женщинами с точки зрения психологии. Поведение настраивается через переменные окружения.

## Возможности

- Читает сообщения в группе и отвечает психологическими подсказками участницам из списка `WOMEN_USER_IDS`.
- Команда `/tip` выдаёт подсказку по запросу.
- Защита от спама через лимит времени между ответами.

## Быстрый старт

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Добавьте бота в групповой чат и дайте ему право читать сообщения.
3. Запустите бота:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

export BOT_TOKEN="ваш_токен"
export WOMEN_USER_IDS="123456,234567"
export MIN_REPLY_SECONDS="3600"  # необязательно

python bot.py
```

## Деплой на Railway

1. Создайте новый проект в Railway и подключите репозиторий.
2. В разделе Variables добавьте:
   - `BOT_TOKEN`
   - `WOMEN_USER_IDS`
   - `MIN_REPLY_SECONDS` (по желанию)
3. Railway автоматически установит зависимости и запустит процесс из `railway.toml`.

## Настройки

- `BOT_TOKEN` — токен Telegram-бота.
- `WOMEN_USER_IDS` — список ID участниц через запятую (например: `123,456`).
- `MIN_REPLY_SECONDS` — минимальная пауза между ответами одному человеку (по умолчанию 3600 сек).

## Как узнать ID пользователя

Попросите участниц написать `/start` вашему боту в личном чате или используйте любого бота-утилиту, который показывает user ID. Затем добавьте ID в `WOMEN_USER_IDS`.

## Этические рекомендации

Психологические подсказки носят общеобразовательный характер и не заменяют консультацию специалиста.
